Projetos

SIGED (SESC/DF)

.NET · C#

Sistema de gestão documental desenvolvido para o SESC/DF. Ao trabalhar lá desenvolvi um sistema de templates de documentos onde os usuários poderiam cadastrar templates para serem utilizados em todos os documentos daquele mesmo tipo. Além disso eu desenvolvi um sistema de permissões e restrições dentro do sistema, visto que eles trabalhavam com documentos muitas vezes sigilosos que eram liberados so para determinadas areas ou determinadas pessoas. Também fiz várias outras evoluções no sistema de acordo com que iam surgindo necessidades do cliente.

SINJ (TCDF)

.NET · C#

Sistema de gestão e divulgação de leis, portarias, decretos e outras normas jurídicas. Enquanto estive encarregado dele fui responsável por dar manutenção e realizar uma serie de evoluções. Desenvolvi um bot que busca e salva diariamente os diarios gerados pelo DODF. Implementei a criação dinamica de relatórios e o envio desses relatórios por email. Além de ter atuado em diversas correções e mudanças de regra pontuais no sistema.

Web Info (PGFN)

Spring · Java

API desenvolvida com a finalidade de unificar e gerenciar permissões de todas as outras APIs e sistemas utilizados pela PGFN.

SISPG/SUSEP (PGFN)

AngularJS · .NET · C#

Sistema de acompanhamento e controle de entregas desenvolvido pela SUSEP. Inicialmente não se adequou ao ambiente da PGFN e fiquei responsável por dar manutenção e evoluir o sistema para atender as necessidades dos gestores.

Sistemas Legados (PGFN)

Java · JSP · JPA

Durante minha estadia na PGFN fiquei responsável por dar manutenção e fazer pequenas evoluções em uma serie de sistemas legados. Dentre eles o Assinatura, o Geplanes, o Permuta e o Promoção.

WS-Log e Library-Log (SESC/MG)

Spring · Java · Kafka

Estrutura de log dos microsserviços do SESC/MG que consiste em um microsserviço (ws-log) que consome mensagens enviadas por todos os outros microsserviços através de uma biblioteca (library-log) que intercepta as requisições e envia ao sistema de mensageria kafka para registro dos logs.

WS-SCA (SESC/MG)

Spring · Java · Kafka

Microsserviço que fornece end-points para serem consumidos pelo Portal de Serviço do SESC/MG.

WS-APP (SESC/MG)

Spring · Java · Kafka

Microsserviço que fornece end-points para serem consumidos pelo aplicativo de celular que está sendo desenvolvido para atender o SESC/MG.

GENB (LightBase)

NestJS · Node

Sistema de gestão e pesquisa de registros de forma relacional e não relacional. Nele fui responsável por desenvolver a parte de pesquisa dos registros.

Portal TCESP (TCESP)

React Native

Aplicativo mobile fornecido pelo TCESP. Nele fui responsável por implementar o Gestor Municipal, um modulo que organiza e apresenta todas as informações dos municipios de São Paulo, como receitas/despesas, indicadores, alertas da LRF dentre outros. Além disso criei um modulo voltado ao EPCP integrado ao moodle, facilitando o acesso de funções básicas do moodle diretamente pelo aplicativo e divulgando os outros meios de comunicação do EPCP. Também criei o modulo de legislações onde o usuario tem acesso a todas as leis, regimentos e documentos juridicos gerados pelo TCESP.

Audesp Web (TCESP)

AngularJS · Spring · Java

Sistema de gestão de auditorias. Enquanto estive nele trabalhei em diversas evolutivas. Criei a homepage com um carrossel e uma lista alertando pendencias e informando prazos. Desenvolvi também um sistema de alteração de Ajustes e Termos Aditivos versionado onde a pessoa fazia uma solicitação ao fiscal e após a aprovação ela poderia modificar e caso o fiscal vetasse a alteração ou então o prazo de alteração passasse da data todas as modificações eram descartadas. Além disso fiz algumas manutenções e evoluções pontuais.
